# ilbar

A simple classic taskbar for Wayland desktops.

## Build requirements

- make
- wayland-client
- wayland-protocols
- wayland-scanner
- A C compiler

Run `make`, and if successful, the executable is located at `build/ilbar`.

## Usage requirements

All you need to run ilbar is a Wayland compositor with layer-shell support.
